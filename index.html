Strider

Brilliant Continuous Deployment

Strider is an Open Source Continuous Deployment / Continuous Integration platform. Strider is conceptually similar to Travis-CI or Jenkins with some major differences.

Github built-in

With native Github integration, you can have your tests automatically run and emailing results on every checkin in a couple of clicks. No messing with SSH keys or fiddly command line scripts needed.

Continuous Deployment: Not just CI

Heroku automation support out of the box, no scripting necessary.

Native Selenium/Sauce Labs support

Get your Selenium/Sauce Labs tests running in parallel across a huge number of browser/OS combinations in less than a minute.

Run a private Strider instance...

On a free Heroku account or your own infrastructure. It's simple to deploy!

But wait, there's more!

  • An emphasis on extensibility. Plugins are powerful, easy to write and simple to install
  • Native Node.JS support
  • Collaborators feature enables teams of developers to share access to projects
  • Custom environment variables let you store project config outside of version control
  • Webhooks feature allows you to run flexible, custom deployment processes
  • Websockets stream test and deploy output in real-time
  • Responsive design so Strider renders nicely on mobile browsers
  • Python/Django, Ruby/Rails and more coming soon!
  • Commercial support, consulting and hosting available from BeyondFog, Inc
  • Written in Node.JS / JavaScript and uses MongoDB as a backing store
