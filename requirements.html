Requirements

Make sure the following dependencies are installed before installing Strider. Each link below will have installation instructions for your OS.

  • nodejs >= 4.2
  • git >= 2.0
  • mongodb (local or remote)
  • node-gyp

MongoDB

MongoDB can be installed locally or you can use a service like mLab.
